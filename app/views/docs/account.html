POST /account/profile

http://api.punditly.com/1/account/profile

updates the properties of the authenticating user's account.

Parameters
name optional Full name associated with the profile. Maximum of 20 characters.
location optional The city or country describing where the account user is located. maximum of 30 characters.
url optional URL associated with the profile. Maximum of 100 characters.
bio optional A description of the user owning the profile. Maximum of 140 characters.

Example Request

POST http://api.punditly.com/1/account/profile
POST Data name=deji%20faletti&bio=Azonto%20Master
Response

{
  "response": {
    "status": 200
  },
	"data":{
	  "bio": "Azonto Master",
	  "id": "52daf3421f31a7e954000001",
	  "location": "atlanta",

	  "url": "punditly.com",

	  "following_count": 110,
	  "followers_count": 20,
	  "is_private": false,
	  "created_at": "2014-01-18T21:33:54.366Z",
	}
}
