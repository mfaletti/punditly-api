GET /friendships/following

http://api.punditly.com/1/friendships/following

returns a list of users the authenticated user is following. Results are given in groups of 20 users and "pages" of results can be navigated through using the since_id and max_id values in subsequent requests.

Parameters
since_id optional Returns results with an ID greater than (more recent than) the specified ID
max_id optional Returns results with an ID less than (older than) or equal to the specified ID

Example Request

POST http://api.punditly.com/1/friendships/following

POST /friendships/create

http://api.punditly.com/1/friendships/create

Allows the authenticating user to create a friendship with the user specified by the id parameter. Returns the befriended user when successful

Parameters
id required The User ID of the user you want to follow.

Example Request

POST http://api.punditly.com/1/friendships/create
POST Data id=1234
